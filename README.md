# ねいろんの使いかた

> すべてのコマンドには **サブプレフィクス** が付きます（例: `!play01 !play02...`）。

---

## サフィックスについて

* テキストコマンドの末尾には `サブプレフィクス` が付与されます。

  > 「GLbot は `!play` だけで動作します。後ろの数字は必要ありません」
* Bot番号が `01` の場合は、 **`!play01`** になります。
* **特例**: `join` はあまり使わないため `join0000` となり、サブプレフィクス込みだと **`!join000001`** になります。

> 画面上のボタン（再生／停止／スキップなど）は **コマンド入力不要**。サーバーに出るUIを押してください。

---

## コマンド一覧

| コマンド        | 使い方                       | 説明                                               |
| ----------- | ------------------------- | ------------------------------------------------ |
| `!join0000` | `!join0000`               | あなたが入室中のボイスチャンネルにBotを接続します（通常は使わない想定）。           |
| `!leave`    | `!leave`                  | Botをボイスチャンネルから切断します。                             |
| `!play`     | `!play <URL または キーワード>`   | YouTube音源を **すぐに再生** します。失敗する場合はURL非対応や制限の可能性あり。 |
| `!stop`     | `!stop`                   | 再生を停止し、キューをクリアします。                               |
| `!playlist` | `!playlist`               | **自分の/サーバーのプレイリスト** を選んで → プレイリスト名を選んで → 曲を操作。   |
| 〃           | `!playlist <name>`        | その名前のプレイリストを表示（ユーザーとサーバー両方にあれば選択UI）。             |
| 〃           | `!playlist user <name>`   | ユーザー側のプレイリストを開く（無ければ作成UI）。                       |
| 〃           | `!playlist server <name>` | サーバー側のプレイリストを開く。                                 |
| `!love`     | `!love`                   | **お気に入り** を表示。UIから再生やシャッフル可能。                    |

---

## 再生UI（ボタン操作）

曲を再生するとメッセージに操作UIが出ます。ボタンは **押した後に無効化** されることがあります。

* **再生／再開** … 一時停止中なら再開。
* **一時停止** … 再生を一時停止。
* **スキップ** … 現在の曲をスキップ。
* **停止** … 再生を止めてキューをクリア。
* **キュー表示** … キュー一覧を表示し、先頭25件まで削除可能。
* **＋追加 / －削除** … ミニ選択UIで以下から選べます：

  * ❤️ お気に入り（ユーザー単位）
  * 👤 ユーザープレイリスト（どこのサーバーでも共通）
  * 🌐 サーバープレイリスト（同じサーバーの利用者なら誰でも再生可）

---

## プレイリスト表示UI（マルチプレイリスト対応）

`!playlist`, `!palylist`, `!love` などで出る一覧では：

* **▶ 再生（選択）** … 選んだ曲をキューへ追加して再生。
* **🔀 シャッフル再生（全曲）** … 全曲をシャッフルしてキュー投入。
* **📂 プレイリスト名選択** … ユーザー/サーバー → プレイリスト名を選んで開く。新規作成も可能。

> プレイリストは複数作成できます。名前が被った場合はユーザー/サーバーどちらで開くか選択UIが出ます。
> リストは25件まで選択対象。残り件数はフッターに表示されます。

---

## 歌詞の表示
* 同期歌詞があれば曲に合わせてスクロール表示。テキストのみならページ分割されます。
* 再生メッセージには「📝 歌詞の間違いを報告」ボタンが表示され、GitHubのIssueから報告可能です。
