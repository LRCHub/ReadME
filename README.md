# ねいろんの使いかた

> すべてのコマンドには **サブプレフィクス**が付きます（例: `!play01 !play02...`）。

---

## サフィックスについて

* 本Botのテキストコマンドは末尾に `サブプレフィクス` が付与されます。
 > 「GLbotは!playのみです。後ろの数字は必要ありません」
* Bot番号が `01` の場合は、  **`!play01`** となります。
* **特例**: `join` はあまり利用しないため `join0000` となります。つまりサブプレフィクスを含めると **`!join000001`** になります。

> 画面上の各種ボタン（再生／停止／スキップなど）は **テキストコマンドではなくUI操作** です。サーバーでメッセージに表示されるボタンを押して操作してください。

| -------------- | ------------------------- | ---------------------------------------------------------------------- | --------------------------------------- |

| `!stop`      | `!stop`                 | 再生を停止し、キューをクリアします。                                                     |                                         |
| `!playlist`  | `!playlist`             | **あなたのプレイリスト** と **サーバープレイリスト** を連続表示します（UIで曲の選択・再生が可能）。               |                                         |
| 〃              | `!playlist user`        | あなたのプレイリストのみ表示。                                                        |                                         |

| 〃              | \`!playlist01 add <曲名     | URL>\`                                                                 | 現在再生中の曲、または指定の曲を **あなたのプレイリストに追加** します。 |

## 再生UI（ボタン操作）

楽曲を再生すると、メッセージに操作UIが表示されます。ボタンは **押下後に無効化** される場合があります（多重投下防止）。

* **再生／再開** … 一時停止中なら再開します。
* **一時停止** … 再生を一時停止します。
* **スキップ** … 現在の曲をスキップし、次の曲へ。
* **停止** … 再生を停止し、キューをクリア。
* **キュー表示** … 現在の再生キューを表示。必要に応じて先頭25件から削除できます。
* **＋ 追加 / － 削除** … 以下をミニ選択UIで切替え可能：

  * ❤️ **お気に入り**（ユーザー単位）
  * 👤 **ユーザープレイリスト** → どこのサーバーにいても同期されます。
  * 🌐 **サーバープレイリスト** → サーバーの利用者なら誰でも再生できます。

---

## プレイリスト表示UI

`!playlist`, `!love` などで表示される一覧には、以下の操作が用意されています。

* **▶ 再生（選択）** … 表示中の一覧から選択した曲をキューへ追加して再生。
* **🔀 シャッフル再生（全曲）** … 一覧の全曲をシャッフルしてキュー投入。

> リストは先頭25件まで選択対象として表示されます。残り件数はフッターに表示されます。
> （Discordの使用による制限）

---

## 歌詞の表示（利用者向けメモ）

* 再生開始を最優先し、**歌詞は後追いで読み込み** ます。表示まで少し時間がかかる場合があります。
* 同期歌詞が見つかると、再生に合わせてスクロール表示されます。テキストのみの場合は複数ページに分割されル場合があります。
